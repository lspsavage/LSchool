Tailwind CDN

CDN (Content Delivery Network) Tailwind adalah cara untuk menyertakan Tailwind CSS ke dalam proyek Anda tanpa perlu menginstalnya melalui package manager seperti npm. Dengan menggunakan CDN, Anda bisa langsung menggunakan Tailwind CSS dalam proyek Anda hanya dengan menambahkan link ke file CSS-nya di HTML. Untuk lebih lanjut mengenai Play CDN.

1. Menambahkan CDN Tailwind

Anda hanya perlu menambahkan tag <link> ke dalam <head> dari file HTML Anda 

                
                
<!DOCTYPE html>
<html lang="en">
  <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Tailwind Project</title>
  <link href="https://cdn.jsdelivr.net/npm/tailwindcss@3.3.0/dist/tailwind.min.css" rel="stylesheet">
</head>
<body>
  <div class="text-center text-blue-500">
    <h1>Hello, Tailwind CSS!</h1>
  </div>
</body>
</html>
