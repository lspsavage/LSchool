Installasi Tailwind dengan Post CSS

1. Buat Folder

Buatlah folder dengan nama "Tailwind" atau apapun untuk menyimpan file-file yang dibuatuhkan

2. Buka Visual Code Studio

Bukalah aplikasi visual code studio editor

3. Buat file tailwind.html

Buat file tailwind.html dan tulis code seperti berikut

              

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
<h1 class="text-3xl font-bold underline"> Hello world!</h1>
</body>
</html>

4. Install Extensions

Masuk ke bagian extensions, install Tailwind CSS IntelliSense dan PostCSS Language Support.

5. Masuk Terminal

Masuk ke bagian menu Terminal > New Terminal.

6. Inisiasi Project

Ketikkan pada terminal npm init -y untuk menginisiasi project

            

npm init -y

7. Install Tailwind

Ketikkan pada terminal npm i -D tailwindcss postcss autoprefixer untuk menginstall Tailwind

            

npm i -D tailwindcss postcss autoprefixer

8. Buat file tailwind.config.js

Ketikkan pada terminal npx tailwindcss init untuk membuat file konfigurasi tailwind yaitu tailwind.config.js

            

npx tailwindcss init

9. Ubah tailwind.config.js

Buka tailwind.config.js untuk mengkonfigurasi konten seperti berikut

            

/** @type {import('tailwindcss').Config} */
module.exports = {
 content: ['./tailwind.html'],
 theme: {
  extend: {},
 },
 plugins: [],
}

10. Buat file tailwind.css

Buatlah file css tailwind.css , lalu tambahkan code berikut

            

@tailwind base;
@tailwind components;
@tailwind utilities;

11. Buat file css akhir

Buat file css akhir yang akan digunakan oleh website kita. Ketikkan pada terminal

            

npx tailwindcss -i ./tailwind.css -o ./output.css --watch

12. Koneksi HTML dengan CSS output

Buka file tailwind.html lalu tambahkan code berikut untuk mengkoneksikan HTML dengan CSS output dari Tailwind. Ubah file Tailwind.html menjadi seperti ini

            

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="output.css">
<title>Tailwind CSS</title>
</head>
<body>
<h1 class="text-3xl font-bold underline"> Hello world!</h1>
</body>
</html>
