Tailwind Laravel

Laravel adalah framework PHP open-source yang dirancang untuk mempermudah pengembangan aplikasi web dengan menyediakan berbagai fitur seperti routing, middleware, dan ORM (Object-Relational Mapping) dengan Eloquent. Laravel mengikuti pola arsitektur MVC (Model-View-Controller) dan menawarkan berbagai alat serta pustaka untuk membantu pengembangan aplikasi yang bersih dan Tailwind CSS adalah framework CSS utility-first yang memungkinkan Anda untuk membangun desain yang dapat dikustomisasi dengan cepat dan mudah menggunakan kelas-kelas utility. Untuk lebih lanjut mengenai Install Tailwind CSS - Laravel.

    Persyaratan:
  • PHP (versi 8.0 atau lebih tinggi)
  • Composer (pengelola dependensi PHP)
  • Server database seperti MySQL, PostgreSQL, SQLite, atau SQL Server.

    • 1. Install Laravel

    Pastikan Anda telah menginstal Composer. Kemudian, jalankan perintah berikut untuk membuat proyek Laravel baru 

                    
                
composer create-project --prefer-dist laravel/laravel nama-proyek

    2. Ubah direktori project

    Setelah install laravel, Anda bisa langsung mengubah direktori anda ke project laravel yang telah dibuat. 

                    
                
cd nama-proyek

    3. Instal Tailwind CSS

    Instal Tailwind CSS dan dependensinya menggunakan npm (Node Package Manager). Setelah itu, jalankan perintah untuk membuat file konfigurasi tailwind.config.js dan postcss.config.js. 

                    
                
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

    4. Konfigurasi Path Template

    Tambahkan path ke file template Anda dalam file konfigurasi Tailwind tailwind.config.js. Ini memastikan Tailwind dapat memproses semua file yang Anda buat. Edit file tailwind.config.js menjadi seperti berikut.

                    
                
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./resources/**/*.blade.php",
    "./resources/**/*.js",
    "./resources/**/*.vue",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

    5. Tambahkan Direktif Tailwind ke CSS

    Tambahkan direktif Tailwind ke file CSS Anda (app.css). Ini akan memastikan bahwa Tailwind CSS dimuat dengan benar. Edit file resources/css/app.css dan tambahkan baris berikut.

                    
                
@tailwind base;
@tailwind components;
@tailwind utilities;

    6. Mulai Proses Build

    Jalankan perintah berikut untuk memulai proses build. Ini akan mengkompilasi file CSS Anda dan mempersiapkannya untuk digunakan di proyek Anda.

                    
                
npm run dev

    7. Gunakan Tailwind di Proyek Anda

    Pastikan file CSS yang telah dikompilasi dimuat di <head> halaman HTML Anda. Kemudian, Anda bisa mulai menggunakan kelas utilitas Tailwind untuk styling konten Anda.Edit file Blade template Anda (misalnya resources/views/app.blade.php) seperti berikut. 

                    
                
<!DOCTYPE html>
<html lang="en">
  <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Tailwind Project</title>
  @vite('resources/css/app.css')
</head>
<body>
  <div class="text-center text-blue-500">
    <h1>Hello, Tailwind CSS!</h1>
  </div>
</body>
</html>

    8. Edit File Routes

    Di Laravel, routing mengatur bagaimana permintaan HTTP diarahkan ke kontroler atau view tertentu. Semua route Laravel didefinisikan dalam file di direktori routes. File utama untuk route web adalah routes/web.php. Anda dapat menambahkan atau mengganti route di sini.

                    
                
// routes/web.php

// Route baru yang menggantikan route default
Route::get('/', function () {
    return view('app');
});

    9. Jalankan Server Laravel

    php artisan serve adalah perintah untuk menjalankan server pengembangan Laravel. Ini memungkinkan Anda untuk mengakses aplikasi Laravel Anda melalui browser tanpa memerlukan server web seperti Apache atau Nginx. Di terminal, jalankan perintah berikut untuk memulai server. 

                    
                
php artisan serve

    10. Akses Aplikasi

    Buka browser dan kunjungi http://localhost:8000 untuk melihat aplikasi Laravel Anda. Jika Anda telah mengganti route, pastikan untuk memeriksa apakah perubahan tersebut terlihat di browser.
